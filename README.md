# HTML `<img>` Optimazation

## 요구사항

1. avif 이미지를 제공하세요. webp, jpeg 이미지를 대체 수단으로 제공하세요.
2. 960픽셀 미만 해상도에서 small 이미지를 출력하세요. 961픽셀 이상 해상도에서 large 이미지를 출력하세요.
3. HTML 코드만을 사용하고 1, 2 요구 사항을 동시에 대응하는 하나의 <picture> 요소를 구현하세요.
